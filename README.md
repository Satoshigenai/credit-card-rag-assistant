# 🏦 Credit Card RAG Assistant

## Version

Version 2.0

## Project Overview

The Credit Card RAG Assistant is an AI-powered banking prototype designed to answer credit card related questions using trusted source documents.

The project uses Retrieval-Augmented Generation (RAG) to retrieve relevant information from uploaded credit card documents and generate context-aware answers.

The objective of this project is to demonstrate how Artificial Intelligence can improve knowledge retrieval, customer service support, and operational efficiency in banking and cards operations.
